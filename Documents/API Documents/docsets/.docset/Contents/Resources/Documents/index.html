STORES Payments SDK

必要条件

  • Xcode: 14.0.1
  • Swift: 5.7.0
  • iOS 13〜

組み込み方法

セットアップ

SDKをProjectに追加する

STORESPaymentsSDK.xcframework を SPM (Swift Package Manager) でProjectに追加してください。
その際にTCMPayment.xcframework も追加してください。

Step1
  • Projectの設定 > Package Dependencies タブ で + をクリック

Step2
  • Search fieldに git@github.com:Coiney-SDK/STORESPaymentsSDK-iOS.git または、 https://github.com/Coiney-SDK/STORESPaymentsSDK-iOS と入力し、 STORESPaymentsSDK-iOS を選択して Add Package ボタンをクリック

Step3
  • STORESPaymentsSDK と TCMPayment 両方にチェックを入れて Add Packageボタンをクリック

Info.plist への追加

BluetoothでSTORES 決済端末に接続し、ICや磁気カード決済をするために、UISupportedExternalAccessoryProtocols をInfo.plistへ追加する必要があります。

<key>UISupportedExternalAccessoryProtocols</key>
<array>
  <string>com.coiney.Coiney</string>
</array>

Bluetoothでの接続の許可を取得する為に NSBluetoothAlwaysUsageDescription 及び NSBluetoothPeripheralUsageDescription を Info.plistに追加します。

<key>NSBluetoothAlwaysUsageDescription</key>
<string>{Bluetooth接続のパーミッション取得時に表示される任意の文字列}</string>
<key>NSBluetoothPeripheralUsageDescription</key>
<string>{Bluetooth接続のパーミッション取得時に表示される任意の文字列}</string>

決済処理で位置情報を取得する必要がある為 NSLocationWhenInUseUsageDescription を Info.plistに追加します。

<key>NSLocationWhenInUseUsageDescription</key>
<string>{位置情報のパーミッション取得時に表示される任意の文字列}</string>

WechatPay 決済を利用する場合は NSCameraUsageDescription を Info.plistに追加します。

<key>NSCameraUsageDescription</key>
<string>{カメラのパーミッション取得時に表示される任意の文字列}</string>

初期化

アプリケーション起動時に configure() メソッドを呼んでください。

※configureメソッドに引数で渡すTokenは、Bundle ID毎に発行が必要です。

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

    func application(
        _ application: UIApplication,
        didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?
    ) -> Bool {

        let token = "{APPLICATION-TOKEN}"
        STORESPaymentsSDK.STP.configure(applicationToken: token) { result in
            print("STP.configure result = \(result)")
        } terminalErrorHandler: { [weak self] terminalError in
            print("\(terminalError.title)")
        }

        return true
    }

設定

決済処理の前に設定を変更することができます。

  • このメソッドを呼ばない場合は以下のデフォルト値で動作します

    • デフォルト値
    • isPrintReceiptEnabled (レシート印刷) : false
    • isSendReceiptMailEnabled (レシートメール送信) : false
    • mode (SDK動作モード) : .staffOperation

  • 設定した値はアプリケーション起動中はSDK側で保持されます。

  • アプリケーション起動中何度でもこのメソッドを使用して設定を変更できます。

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

        let settings = STORESPaymentsSDK.STP.Settings(
            isPrintReceiptEnabled: true,
            isSendReceiptMailEnabled: false,
            mode: .staffOperation)
        )
        STORESPaymentsSDK.STP.setup(settings)

ログイン

STORES 決済にログインします

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

    STORESPaymentsSDK.STP.login(presentOn: self) { [weak self] result in
        switch result {
        case .success:
            print("Login Success!")
        case let .failure(error):
            print("Login Error! \(error)")
        }
    }

決済

決済画面を表示します

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

    STORESPaymentsSDK.STP.payment(
        presentOn: self,
        amount: 1000,
        memo: "Memo") { [weak self] transactionType in
            print("Payments Finish \(transactionType)")
    } completion: { [weak self] result in
        switch result {
        case let .success(transactionType):
            print("Payments Complete \(transactionType)")
        case let .failure(error):
            print("Payments Error \(error)")
        }
    }

売り上げ情報・返金

売り上げ情報取得

売り上げ情報を取得します

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

    let transactionId = "{Transaction_ID}"
    STORESPaymentsSDK
        .STP
        .lookUpTransaction(transactionIdentifier: transactionId) { [weak self] result in
            switch result {
            case let .success(transactionType):
                print("Lookup transaction success \(transactionType)")
            case let .failure(error):
                print("Lookup transaction error \(error)")
            }
        }

売り上げ詳細画面表示

売り上げ詳細画面表示を表示します
この画面から返金処理も実行できます

また、shouldAutoTransitionToRefund を true にすることで
自動的に返金画面へ遷移します

実装例

import STORESPaymentsSDK

〜〜 中略 〜〜

    let transactionId = "{Transaction_ID}"
    STORESPaymentsSDK
        .STP
        .presentTransactionViewController(
            on: self,
            transactionIdentifier: transactionId,
            shouldAllowRefunding: true,
            shouldAutoTransitionToRefund: false,
            refundErrorHandler: { error in
              print("Refund Error = \(error)")
            },
              dismissCompletion: { result in
                switch result {
                case let .success(transactionType):
                    print("Show transaction success \(transactionType)")
                case let .failure(error):
                    print("Show transaction error \(error)")
                }
            })
